## Unity UI Toolkit Documentation

FM Skin Builder works with Unity's UI Toolkit (USS/UXML). These official Unity docs are helpful:

### Unity 6 (Current)

- [UI Toolkit Overview](https://docs.unity3d.com/6000.3/Documentation/Manual/UIElements.html) - Introduction to UI Toolkit
- [USS Properties Reference](https://docs.unity3d.com/6000.3/Documentation/Manual/UIE-USS-Properties-Reference.html) - All supported USS properties
- [USS Selectors](https://docs.unity3d.com/6000.3/Documentation/Manual/UIE-USS-Selectors.html) - Selector syntax guide
- [USS Data Types](https://docs.unity3d.com/6000.3/Documentation/Manual/UIE-USS-PropertyTypes.html) - Color, length, and other value types
- [UXML Reference](https://docs.unity3d.com/6000.3/Documentation/Manual/UIE-UXML.html) - UXML element reference
- [Built-in USS Variables](https://docs.unity3d.com/6000.3/Documentation/Manual/UIE-USS-UnityVariables.html) - Unity's default variables

### Additional Resources

- [MDN CSS Reference](https://developer.mozilla.org/en-US/docs/Web/CSS) - Standard CSS reference (USS is similar)
