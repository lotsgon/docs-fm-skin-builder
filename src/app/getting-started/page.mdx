## Prerequisites

Before you begin, make sure you have:

- **Football Manager 26** installed on your computer
- Basic familiarity with file management (creating folders, editing files)

No programming experience is required! FM Skin Builder is designed to be accessible whether you're writing raw code or using visual editors.

## Quick Path

If you're eager to dive in from scratch:

1. [Download FM Skin Builder](/downloads)
2. Open the app and create a new skin project
3. Make a simple color change in the CSS Editor
4. Build and test in FM26

Or you can start with an existing community skin template:

- [Darkside Neutral Theme](https://sortitoutsi.net/content/75885/darkside-neutral-theme-blue-and-kaiserslautern-legend-edition) - A neutral theme with custom fonts, backgrounds and structured folders.

1. Use a community skin template as your starting point
2. Open it in FM Skin Builder
3. Make your customizations
4. Build and test in FM26

The [Your First Skin](/getting-started/first-skin) guide walks through exactly this process.
